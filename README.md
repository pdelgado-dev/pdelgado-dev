## Digital Order Management System

Full-stack restaurant POS platform built with ASP.NET Core MVC.

### Key Features
- Role-based access control (Admin · Waiter · Kitchen)
- Real-time kitchen communication using SignalR
- REST API consumed by Android mobile app
- JWT Authentication with Cookie + Bearer Token support
- Admin panel for menus, tables, categories, and users

### Stack
`C#` · `ASP.NET Core MVC` · `EF Core` · `SQL Server` · `SignalR` · `Bootstrap 5`
